# Habitat HMD Project Hub

Internal project management tool for **Habitat for Humanity Halton-Mississauga-Dufferin**.

## Features
- Kanban board with drag-and-drop
- Multiple teams and projects
- Task management with priorities, assignees, and due dates
- Resource management (links & files)
- Feedback collection with task conversion
- Project changelog
- Team messaging
- Responsive design (works on desktop, tablet, and mobile)

## Deployment
This app is deployed automatically via Netlify. Push to `main` to trigger a new build.
